## 一个基础的``SSM``整合框架底包

这是一个自用的``SSM``开发底板，将项目``clone``下来继续开发，已经完成Spring、SpringMVC、Mybatis整合,
严格按照代码规范编写，方便继续开发

## 需要工具

- ``idea2020.3.4``
- ``mysql5.7``
- ``Tomcat9``
- ``maven``

## 目录介绍

- ``resources``-配置文件目录
- ``java``-``java``的根目录

## How to use

1. ``clone``项目
```shell
git clone https://github.com/koyangyang/BaseSSM.git
```
2. 用``idea``打开
3. 用``maven``加载依赖``jar``包
4. 运行``web``项目,打开``http://localhost:8080/test``
